# 포스트 반도체, K-컬쳐 수출 주도주 발굴 및 최적 자산 배분
## 1. 프로젝트 개요
이 프로젝트는 반도체 의존에서 벗어나 대한민국의 새로운 경제성장 동력을 데이터로 확인하고, 안정적이며 지속적인 수익창출 전략 수립을 위한 자산 배분 포트폴리오입니다.

## 2. 투자 아이디어
* 내수소멸 : 저출산과 불황에 따른 내수 부진, 전 세계를 대상으로 하는 수출기업의 경쟁력 강화
* 중국 제조업 경쟁력 부상 : 기존과 같은 고품질, 저가 생산품으로는 생존이 어려움
* 변동성 헤지 : 반도체 사이클 의존성을 낮추고 경기 변동성에 강한 소비재이자 고성장주의 필요성 강화

  * PyKRX: 국내 주식 시장의 종가(OHLCV), 시가총액 데이터 수집
